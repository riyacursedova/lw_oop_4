# Лабораторна робота №4

### Тема
**Робота з принципами ООП в С#**

### Мета
Закріпити навички роботи з принципами об'єктно-орієнтованого програмування (ООП) на мові **C#**.

**Основні завдання:**
* Закріпити навички використання **інтерфейсів, наслідування та поліморфізму**.
* Навчитись правильному **розподіленню відповідальності** між класами.
